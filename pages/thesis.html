Bachelor's Thesis

Smart Vehicle Localization System in Parking Areas

completed

Purpose: University thesis project.

Summary: Urbanization has led to an increase in vehicles and increasingly complex parking environments, making it difficult for users to locate their cars in large lots such as those at shopping malls, airports, or public venues. This project proposes an Internet of Things (IoT)-based localization system designed to help users find their parked cars easily.

The system uses HC-SR04 ultrasonic sensors for vehicle detection and ESP32-CAM modules for image capture. License plate recognition is achieved through image processing and optical character recognition using OpenCV and Tesseract OCR. All collected data is stored in a centralized MySQL database. Users can retrieve the location of their vehicle through a dedicated Android application by entering their license plate number.

* The system is currently designed for numbered parking lots, with each space identified by a unique code or number.

Software Components:

  • ParkCam (PC GUI): A Python-based desktop interface built with PySimpleGUI. It displays real-time images and handles tasks such as MQTT communication with AWS EC2, image processing, OCR, API interaction, and database updates. Asynchronous functions and threading are used to manage the simultaneous image streams from multiple cameras, ensuring efficient processing and display.
  • AWS EC2 (Linux): Hosts the Mosquitto MQTT broker to facilitate image transmission between the ESP32-CAM and ParkCam.
  • Find&Drive (Android App): Allows users to register their vehicles and retrieve the location code associated with a license plate.
    The mobile app can be found here.
  • Database & Backend: A MySQL database managed with phpMyAdmin, and PHP-based REST APIs used to handle data requests from the PC app and Android app.
    • Note: Initially, Firebase Realtime Database was used, but later replaced with MySQL to better support structured data and more complex queries.

Technologies: VS Code, PlatformIO, C++ (Arduino), AWS EC2, Mosquitto MQTT Broker, Python, PySimpleGUI, OpenCV, Tesseract OCR, PHP, cPanel, REST APIs, MySQL, phpMyAdmin, Android Studio, Java (Android), Volley Library, Material Design 3.
